# LazyVerilog v1.0.2 Release Notes

Bug-fix release.

## Fixes

- **Hover** — fixed hover resolving to the wrong scope in certain nested contexts.
- **RTL Tree** — fixed server ignoring `[rtltree]` config options (`show_instance_name`, `show_file`).
- **Config** — fixed incorrect default values in `lazyverilog.toml`.
