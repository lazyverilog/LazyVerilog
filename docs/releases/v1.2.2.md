# LazyVerilog v1.2.2 Release Notes

## macOS Compatibility
- Replace std::jthread/stop_token with std::thread/atomic — fixes build on older macOS toolchains
- macOS 14 deployment target for darwin-arm64; use Xcode 16 / macOS 15 runner for C++20 support

## Linux Static Builds
- New release artifacts: linux-x64-static and linux-arm64-static — embed C++ runtime, no glibc runtime dependency
- Neovim plugin automatically falls back to static build if dynamic binary fails ldd check (glibc version mismatch)

## Release Infrastructure
- Automated GitHub Actions release pipeline (replaces manual script)
- Pre-release build test prompt in tools/release.sh
- Release status badge added to README
