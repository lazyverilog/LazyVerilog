# LazyVerilog v1.2.6 Release Notes

Formatter alignment fix and safer binary installs.

## Formatter Fix

- Fixed declaration and port alignment when lines have different combinations of types, packed dimensions, unpacked dimensions, assignments, or delimiters. Formatted declarations now keep stable, readable columns instead of shifting fields left or right unexpectedly.

## Install Security

- Auto-installed `lazyverilog-lsp` release binaries are now verified with SHA-256 checksums before they are used.
