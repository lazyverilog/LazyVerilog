Documentation for the lazyverilog linter. Primary reference for all `lazyverilog.toml` lint options.

## For AI Agents

### Working In This Directory
- When adding a new lint rule in `src/features/lint.cpp` + `src/config.hpp`, add an entry to `options.md`
- Each option should include a toml snippet, a bad/good RTL example, and the exact diagnostic message string
